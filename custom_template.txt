# HOW TO USE
#
# 1. Look up the item code for the item you want in the Appendix A at the end
#       of the file. (ex. Final Sword has an item code "31A")
# 2. Find the location below where you want to put the item. (ex.
#       "item_shortsword_03" is the location of the short sword at the
#       beginning of the game)
# 3. Type the item code next to the chosen location, separated by a space.
#       (ex. "item_shortsword_03 31A" to put a Final Sword at the beginning
#       of the game)
# 4. Repeat for as many items and locations as you want. 
# 5. Run the randomizer, and in the "flags" section type "custom" to activate
#       custom mode.
# 6. Type in the name of this file (including the ".txt" extension) when the
#       randomizer asks for your custom mode filename.
#
# Any unfilled locations will be filled in by the randomizer. The randomizer 
#       will try to make the seed beatable with your custom restrictions.
#       The unfilled locations will be determined by the seed number. If you
#       do something like putting Skula in the Arena, then the seed won't
#       be beatable.
